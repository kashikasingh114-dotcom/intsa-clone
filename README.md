# 📸 ReelGramm – Instagram Clone (Streamlit)

ReelGramm is a simple Instagram-like social media app built using **Python and Streamlit**.  
It allows users to post images, like, comment, follow users, and view profiles.

---

## 🚀 Features

- 🔐 Login / Signup system
- 🏠 Instagram-style feed
- ❤️ Like posts
- 💬 Comment system
- ➕ Upload posts with images + captions
- 👤 Profile page with DP upload & delete
- 👥 Followers & Following system
- 🔍 User search functionality

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- JSON + CSV (for storage)  
- Local file system (images)
